Site Report

1. Design Choices

design of this website focuses on a clean, user-friendly layout and visual consistency.

  • Color Palette: Minimalistic grayscale with subtle accents for a professional look.
  • Font: ‘Poppins’ for its modern and approachable aes tic.
  • Navigation: Sticky navbar for better usability and a responsive menu for hamburger mobile devices.
  • Animations: Subtle fade-ins and transitions enhance user engagement without overwhelming.
  • Grid Layout: Ensures proper alignment and responsiveness across devices.

2. Development Challenges

Key challenges faced and solutions implemented:

  • Responsive Layout: Switched to CSS grid with `minmax` for better adaptability on small screens.
  • Mobile Navigation: Implemented a CSS-only hamburger menu using a hidden checkbox.
  • Hover Effects: Experimented with box shadows and transitions for balanced interactivity.
  • Form Validation: HTML required attributes were used as a temporary solution, with plans for JavaScript validation.

3. Future Improvements

  • Dark Mode: Adding a toggle for improved user experience in low-light environments.
  • Project Details: Including a “read more” link or modal for detailed descriptions.
  • Enhanced Validation: Implementing JavaScript form validation for more robust user feedback.

4. Validation Images

  • HTML and CSS Validation: Screenshots of validation reports to showcase compliance with W3C standards.

5. Conclusion

This project helped me learn responsive design, CSS animations, and importance of usability. It serves as a portfolio piece and foundation for future projects.
